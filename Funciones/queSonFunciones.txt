// Diferencia entre Parametros y Argumentos.
// La variable que envíamos a la función se le llama argumento (desde el programa main)

// Diferencia entre Procedimiento y Funcion.
// un procedimiento no devuelve ningun valor, puede mostrar la salida a la consola
// pero no devuelve ningun valor, así como si lo hace una función.

    -Funciones definidades por el usuario
